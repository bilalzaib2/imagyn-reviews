# Contributing

## Mission

Imagyn Reviews should feel like a premium SaaS product.

Quality is more important than speed.

## Before Coding

Read:

- README.md
- DESIGN_PRINCIPLES.md
- DEVELOPMENT_GUIDE.md

## Pull Requests

Every change should:

- improve maintainability
- improve readability
- avoid duplication

## Never

- Commit broken code
- Duplicate components
- Hardcode values
- Ignore accessibility

## Always

Think before coding.
