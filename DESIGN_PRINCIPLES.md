# Imagyn Reviews Design Principles

## Philosophy

Imagyn Reviews should feel premium, elegant, calm, and effortless.

The interface should resemble products like Apple, Linear, Notion, Arc Browser, and Raycast.

Avoid clutter.

Whitespace is a feature.

## Visual Style

- Minimal
- Modern
- Soft shadows
- Large spacing
- Rounded corners
- Elegant typography
- Subtle animations
- Premium feeling

## Color

Mostly neutral colors.

White backgrounds.

Light gray borders.

Black typography.

One primary accent color only.

No bright gradients.

No unnecessary illustrations.

## Components

Every component must be reusable.

Cards should be simple.

Buttons should feel premium.

Forms should be clean.

Tables should be spacious.

## UX

Everything should require the fewest clicks possible.

No overwhelming dashboards.

Prioritize clarity over density.
